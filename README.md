Responsive Personal Portfolio Website
A responsive personal portfolio website built with HTML, CSS, and JavaScript.

Features
- Smooth scrolling between sections.
- Animations triggered on scroll for a dynamic user experience.
- Fully responsive design for all screen sizes.
- Customized layout and styling to showcase personal projects and experience.

Technologies Used
- HTML5
- CSS3 (with animations and responsive grid/flex layouts)
- JavaScript (for interactivity, form validation, and scroll effects)
